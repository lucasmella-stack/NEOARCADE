# Política de Seguridad

## Versiones soportadas

| Versión | Soportada |
| ------- | --------- |
| 0.1.x   | ✅        |

## Reportar una vulnerabilidad

Si encontrás una vulnerabilidad de seguridad en NEOARCADE, **no abras un issue público**.

En su lugar:

1. Enviá un email a **security@neoarcade.dev** (o contactá al maintainer directamente por GitHub).
2. Incluí una descripción detallada de la vulnerabilidad.
3. Si es posible, incluí pasos para reproducirla.

Vas a recibir una respuesta dentro de 48 horas con los próximos pasos.

## Qué reportar

- Vulnerabilidades de inyección (XSS, command injection, etc.)
- Bypass de controles de acceso
- Exposición de datos sensibles
- Cualquier comportamiento inesperado que pueda ser explotado

## Qué NO reportar como vulnerabilidad

- Bugs funcionales (usá la plantilla de bug report)
- Feature requests
- Problemas de rendimiento
