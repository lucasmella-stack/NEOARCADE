# Changelog

Todos los cambios notables de este proyecto se documentan en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.1.0/),
y el proyecto adhiere a [Semantic Versioning](https://semver.org/lang/es/).

## [0.1.0] - 2026-03-19

### Agregado

- Pantalla principal con 4 juegos JS integrados (Snake, Pong, Breakout, Space Invaders)
- Soporte para ROMs vía EmulatorJS (NeoGeo, SNES, NES, Arcade)
- Controlador móvil con D-pad, botones A/B, Start/Select
- Estética retro 3D (cuerpo azul, botones rosa/amarillo, glow cyan)
- Conexión QR: escanear desde el móvil para conectarse como joystick
- Soporte 2 jugadores simultáneos vía Socket.io
- Fullscreen + landscape lock automático (Android e iOS Safari)
- Lobby con indicadores de conexión por jugador
- Selector de core (emulador) desde la barra superior
- Carga de ROMs por el usuario desde la interfaz
